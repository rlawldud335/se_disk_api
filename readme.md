# 프로젝트 소개
학교에서 창의융합설계를 하면서, 예전에 선배들이 진행했던 프로젝트 데모영상이나 산출물들을 웹으로 조회 하면 좋겠다고 생각해서 만들게 됨. 프로젝트에 대해 질문이나 피드백을 할 수 있는 커뮤니티 기능도 필요 추가 함.

## 핵심기능
1. 웹에서 한글, MS office문서, 동영상을 조회하기
2. 키워드나 태그별로 프로젝트를 검색하는 기능
3. 프로젝트 좋아요,댓글 기능
